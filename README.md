## ✨ 功能特性

### 标记系统
- ✅ **20 种标记类型** - 宝箱（7 种）、探索（4 种）、生物（5 种）、其他（4 种）
- ✅ **分类折叠** - 标记类型按分类组织，可折叠/展开
- ✅ **标记聚合** - 缩放时标记自动聚类显示（可开关）
- ✅ **标记备注** - 为每个标记添加自定义备注
- ✅ **自动保存** - 所有操作自动保存到 localStorage

### 多地图支持
- ✅ **3 张地图** - 夏露露村、新芽山谷（单层）、弗利斯
- ✅ **独立存储** - 每张地图的标记数据独立存储
- ✅ **地图切换** - 下拉菜单快速切换地图
- ✅ **视图记忆** - 自动保存每张地图的缩放级别和中心点

### 数据管理
- ✅ **导出所有地图** - 一次性导出所有地图的完整数据
- ✅ **导入多地图** - 支持导入多地图格式的 JSON 文件
- ✅ **分享链接** - Base64 编码，一键分享给好友
- ✅ **批量清空** - 清空当前地图或所有地图的标记

### 筛选显示
- ✅ **分类筛选** - 按分类折叠筛选，快速显示/隐藏
- ✅ **实时统计** - 显示各分类的标记数量
- ✅ **全部切换** - 一键显示/隐藏所有标记

## 📊 使用说明

### 添加标记
1. 在左侧选择标记类型（如"宝箱" → "金宝箱"）
2. 点击地图任意位置
3. 在弹窗中添加备注（可选）
4. 点击"保存"

### 编辑标记
1. 点击已有标记
2. 修改备注内容
3. 点击"保存"

### 删除标记
1. 点击标记
2. 点击"删除"按钮

### 导出数据
1. 点击"导出数据"按钮
2. 选择保存位置
3. 文件名格式：`promilia-all-maps-时间戳.json`
4. **包含所有地图的完整数据**

### 导入数据
1. 点击"导入数据"按钮
2. 选择之前导出的 JSON 文件
3. 确认导入信息（显示每张地图的标记数量）
4. 标记会自动加载到对应地图

### 分享标记
1. 点击"分享标记"按钮
2. 分享链接会自动复制到剪贴板
3. 发送给好友，好友打开链接即可加载
4. **限制：** URL 长度有限制（约 2000 字符），大量标记建议使用导出/导入

### 筛选显示
1. 点击左侧筛选区的分类头部可折叠/展开
2. 点击具体类型图标切换显示/隐藏
3. 蓝色高亮表示显示，灰色表示隐藏
4. 点击"全部显示"可一键切换

### 标记聚合
1. 侧边栏有"启用聚合"开关
2. 启用时：缩放时标记自动聚类
3. 禁用时：所有标记直接显示
4. 开关状态自动保存

## 🔧 自定义配置

### 添加新地图

编辑 `index.html`，找到 `MAP_CONFIGS` 对象：

```javascript
const MAP_CONFIGS = {
    shalulu: {
        name: '夏露露村',
        width: 2048,
        height: 2048,
        image: 'maps/shalulu.png',
        storageKey: 'promilia-markers-shalulu',
        color: '#4a90d9',
        type: 'image'
    },
    // 添加新地图...
};
```

**添加新地图：**
```javascript
newmap: {
    name: '新地图',
    width: 2048,
    height: 2048,
    image: 'maps/newmap.png',
    storageKey: 'promilia-markers-newmap',
    color: '#5cb85c',
    type: 'image'  // 或 'tiles'（瓦片）或 'tileLayer'（多层瓦片）
}
```

然后在 HTML 的 `<select id="map-select">` 中添加选项：
```html
<option value="newmap">🗺️ 新地图</option>
```

### 添加新标记类型

编辑 `index.html` 中的 `MARKER_CONFIGS`：

```javascript
const MARKER_CONFIGS = {
    // === 宝箱类 ===
    chest: { icon: 'fa-gift', color: '#f39c12', label: '宝箱', category: 'chest' },
    chest_gold: { icon: 'fa-box', color: '#ffd700', label: '金宝箱', category: 'chest' },
    
    // === 探索类 ===
    puzzle: { icon: 'fa-puzzle-piece', color: '#a55eea', label: '解谜', category: 'explore' },
    
    // === 生物类 ===
    umi: { icon: 'fa-cat', color: '#2d3436', label: '乌咪', category: 'creature' },
    
    // === 其他 ===
    enemy: { icon: 'fa-skull', color: '#e74c3c', label: '敌人', category: 'other' },
    custom: { icon: 'fa-star', color: '#2ecc71', label: '自定义', category: 'other' }
};
```

然后在 HTML 的标记类型区域添加对应的选择按钮。

## 💾 数据存储

- **位置：** 浏览器 localStorage
- **键名：** `promilia-markers-{mapId}`（每张地图独立）
- **容量：** 约 5-10MB（可存储数千个标记）
- **持久化：** 关闭浏览器后数据保留
- **超时保护：** 地图加载 15 秒超时，确保显示

**注意：** 清除浏览器缓存会删除数据，请定期导出备份！

---

## 🌐 分享功能原理

分享链接使用 Base64 编码将标记数据嵌入 URL：
```
https://your-site.com#index.html#data=eyJtYXJrZXJzIjpbXX0=
```

好友打开链接后，数据会自动解码并加载到地图中。

**限制：** URL 长度有限制（约 2000 字符），大量标记建议使用导出/导入功能。
